# TP Symfony – Formulaire simple

## Description

Ce projet consiste à développer une application web Symfony permettant de gérer un formulaire de commande pour un casque audio premium. L'application utilise les Form Types de Symfony pour créer un formulaire professionnel avec validation côté serveur.

## Objectifs pédagogiques atteints
- Maîtrise des formulaires Symfony (Form Types)
- Personnalisation des vues avec Twig
- Intégration de Bootstrap dans un projet Symfony
- Gestion des validations de formulaire

## Fonctionnalités implémentées

### 1. Formulaire Symfony personnalisé
- Création d'un `ProductOrderType` avec Form Builder
- Champs : quantité (IntegerType) et couleur (ChoiceType)
- Bouton de soumission stylisé avec Bootstrap

### 2. Interface utilisateur
- Design responsive avec Bootstrap 5.3
- Image du produit
- Informations détaillées (prix, description, caractéristiques)
- Formulaire intégré de manière élégante

### 3. Gestion des soumissions
- Traitement des données dans le contrôleur
- Messages flash de confirmation
- Redirection après soumission réussie

### 4. Personnalisation Twig
- Utilisation des helpers de formulaire Twig
- Extension du template de base
- Intégration des assets Bootstrap via CDN

## Objectif pédagogique

Ce tp permet de :

* comprendre les Form Types
* utiliser Twig
* gérer une requête POST
* structurer un petit projet Symfony
